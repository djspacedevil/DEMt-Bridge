# DEMt Bridge (Deutsche eMark TRON Bridge)

**Website:** [https://swap-pool.de](https://swap-pool.de)  
**Contact:** info@swap-pool.de  
**GitHub:** https://github.com/djspacedevil/demt-bridge  

The DEMt Bridge connects the classic Deutsche eMark (DEM) blockchain with the TRON network by issuing a tokenized representation (DEMt) on TRON.  
This repository hosts public documentation, the project whitepaper, and the smart contract source file used for verification.

> ⚠️ The operational backend and agent code are private for security reasons.  
> Only public documentation and contract sources are published here for transparency and verification purposes.

## Files
- **DEMt_Bridge_Whitepaper_EN_DE.pdf** — Official whitepaper (English/German)  
- **DEMtToken.sol** — Smart contract used on TRON Nile testnet  
- **README.md** — Project description and contact information

- **Token Name:** Deutsche eMark TRON
